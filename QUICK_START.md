### Opción 2: Compilar Tú Mismo (5 minutos)

#### Requisitos:
- Windows/Mac/Linux
- Node.js instalado
- Android Studio (opcional)

#### Pasos:

```bash
# 1. Clonar
git clone https://github.com/gueyepapematar2007-star/gym-tracker-app.git
cd gym-tracker-app

# 2. Instalar
npm install

# 3. Compilar APK
cd android
./gradlew assembleRelease
cd ..

# 4. El APK estará en:
# android/app/build/outputs/apk/release/app-release.apk
```

#### Instalar en tu móvil:

1. Conecta tu móvil por USB
2. Copia el APK
3. Abre en tu móvil
4. Dale permiso e instala
5. ¡Listo! 🎉
